# Tâche de Stroop (FR) — PyScript

Ce fichier HTML contient une tâche de Stroop écrite **en Python** et exécutée dans le **navigateur** via PyScript.
Aucune installation n’est nécessaire, seulement une connexion Internet (le fichier charge PyScript via CDN).

## Utilisation
1) Ouvrez `stroop_fr_pyscript.html` dans un navigateur récent (Chrome, Edge, Firefox).
2) Cliquez sur **Démarrer** (ou **Démo**) et répondez avec les touches :
   - R = Rouge, V = Vert, B = Bleu, J = Jaune, N = Noir, P = Violet.
3) À la fin, consultez les statistiques et téléchargez les données en **CSV**.

## Partage / Hébergement
- Glissez‑déposez ce fichier sur Netlify Drop ou publiez-le via GitHub Pages pour obtenir une URL publique.
- Vous pouvez aussi l’ouvrir depuis Google Drive (clic droit > Ouvrir avec > Afficher dans un nouvel onglet) selon les paramètres.

## Vidéo / Démo
- Ouvrez la page et enregistrez l’écran (Loom, OBS, ou l’outil d’enregistrement intégré de votre OS) pour produire une vidéo de la tâche.
